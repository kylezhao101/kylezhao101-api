# All Data

The **All** endpoint allows you to retrieve all information stored in the resume database. This includes educational background, professional experience, and project details.


## Summary Table

| **Method**   | **Endpoint**              | **Requires header `apikey`** | **Requires `:id`** |
|--------------|---------------------------|-----------------------|--------------------|
| <Badge variant="get">GET</Badge>    | `/api/all`           | No                    | No                 |

# Usage 

## Getting all information 

To fetch all data, send a <Badge variant="get">GET</Badge> request to the following URL:

```http
GET /api/all
```

## Details

Each entry in the returned JSON will contain the following fields:

- **education**: An array containing the educational background information.
- **experience**: An array detailing the professional experience.
- **projects**: An array of projects worked on.
